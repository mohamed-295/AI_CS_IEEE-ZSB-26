# YOLO Object Detection — Task 19

Object detection application powered by YOLOv11 and OpenCV

## Features
- **Live Camera Mode:** Real-time detection via webcam feed.
- **Video File Mode:** Processes offline video files frame-by-frame and exports an annotated output video (`annotated_output.mp4`).
- **Real-Time Visualizations:** Displays bounding boxes, confidence scores, and live FPS metrics on screen.
- **Object Counting (Bonus):** Dynamically tracks and displays the count of a specific target object per frame.

## Dataset
- Custom dataset created and augmented using Roboflow.
- Dataset URL: https://app.roboflow.com/mohamed-adel-yne4b/d1-fruits/1 

## Installation & Setup
1- pip install -r requirements.txt

2- python main.py
